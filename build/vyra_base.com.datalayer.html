vyra_base.com.datalayer.interface_factory.create_vyra_callable(name, type, node, callback=None, async_loop=None)¶

Create a callable for a V.Y.R.A. (V.Y.R.A. Operating System) service.

A callable is a function that provides a quick response to the request and does not block the caller. The callable must return a value within a given time limit, otherwise it is considered a failure.

Parameters:

  • name (str) – Name of the callable.

  • type (Any) – The ROS2 service datatype definition.

  • node (VyraNode) – The ROS2 node definition.

  • callback (Callable or None) – Callback function to be called when the service is invoked.

  • async_loop (Any) – Optional event loop for asynchronous execution.

Raises:

ValueError – If no callback function is provided.

Returns:

The created VyraCallable object.

Return type:

VyraCallable

vyra_base.com.datalayer.interface_factory.create_vyra_job(name, type, async_loop=None)¶

Create a job for a V.Y.R.A. (V.Y.R.A. Operating System) service.

A job is a function that will be executed in the background and will not block the caller. A job can take a long time to complete and will return a result when it is done. Additionally, a job can provide feedback information during the process.

Parameters:

  • name (str) – Name of the job.

  • type (Any) – The ROS2 service datatype definition.

  • async_loop (Any) – Optional event loop for asynchronous execution.

Return type:

None

Returns:

None

Create a speaker for a V.Y.R.A. service.

A speaker is a publisher that sends messages to a topic and is used to publish simple data types to other V.Y.R.A. OS modules.

  • type (Any) – The ROS2 message datatype definition.

  • node (VyraNode) – The ROS2 node definition.

  • periodic (bool) – Whether the speaker should publish periodically.

  • interval_time (float or None) – Interval time for periodic publishing.

  • periodic_caller (Callable or None) – Callable for periodic publishing.

  • qos_profile (int or QoSProfile) – Quality of Service profile.

  • async_loop (Any) – Optional event loop for asynchronous execution.
