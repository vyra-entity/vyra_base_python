vyra_base.extract_ros_interfaces(target_package_path)¶

Extract ROS2 interface files from the pip-installed library into a ROS2 package.

Parameters:

target_package_path (str or pathlib.Path) – Path to the target ROS2 package where interfaces will be extracted.

Returns:

None

Return type:

None

This function copies interface files (msg, srv, action, json) from the installed vyra_base library to the specified ROS2 package directory.
