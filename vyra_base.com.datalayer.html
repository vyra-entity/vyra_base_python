vyra_base.com.datalayer.interface_factory.run_vyra_job(type, node, send_data, ident_name='global_job_runner', async_loop=None)[source]¶

Create a job runner for an external V.Y.R.A. (V.Y.R.A. Operating System) service.

A job is a function that will be executed in the background and will not block the caller. A job can take a long time to complete and will return a result when it is done. Additionally, a job can provide feedback information during the process. This method creates a client that can call an external job service.

Parameters:

  • type (Any) – The ROS2 service datatype definition.

  • node (VyraNode) – The ROS2 node definition.

  • send_data (dict) – Data to be sent to the job service.

  • ident_name (str) – Identifier name for the job.

  • async_loop (Any) – Optional event loop for asynchronous execution.

Returns:

The created VyraJobRunner object.

Return type:

VyraJobRunner
