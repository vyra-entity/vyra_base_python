# Example 11 - security

Contains merged security examples:

- `security_level_matrix.py` (pure Python security-level matrix)
- `access_level_example.py` (decorator/access-level example)
- `secure_server_node.py` (secure server with RequestAccess)
- `secure_client_node.py` (secure client authentication flow)

Run:

```bash
python examples/11_security/security_level_matrix.py
python examples/11_security/access_level_example.py
python examples/11_security/secure_server_node.py
python examples/11_security/secure_client_node.py
```
