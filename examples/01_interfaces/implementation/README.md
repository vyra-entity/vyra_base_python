# Example 01/interfaces/implementation

This folder contains implementation examples for all interface categories:

- Service: `service_server.py`, `service_client.py`
- Message (pub/sub): `publisher.py`, `subscriber.py`
- Action: `action_server.py`, `action_client.py`

## Run examples

```bash
python examples/01_interfaces/implementation/service_server.py
python examples/01_interfaces/implementation/service_client.py
python examples/01_interfaces/implementation/publisher.py
python examples/01_interfaces/implementation/subscriber.py
python examples/01_interfaces/implementation/action_server.py
python examples/01_interfaces/implementation/action_client.py
```

These files are the canonical implementation reference for message, service, and action callbacks.
